# EffattaReceiptsSwift

A lightweight Swift wrapper for the [Effatta Scontrini](https://effatta.it) API.

## Features

- Send receipt to SDI
- Cancel a receipt

## Installation

Use [Swift Package Manager](https://swift.org/package-manager/) to add the dependency:

```swift
.package(url: "https://github.com/Onnwen/effatta-receipts-swift.git", from: "1.0.0")
```

Then add "EffattaReceiptsSwift" as a dependency for your target.
